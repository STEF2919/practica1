FLUTTER

Es un framework de código abierto y gratuito de Google que permite crear aplicaciones nativas para Android e iOS con una base de código sencilla. Se distingue por su nueva forma de crear aplicaciones nativas. Es un marco de interfaz de usuario de smartphone fiable para desarrollar aplicaciones atractivas rápidamente acelerando el desarrollo.

Caracteristicas

  • Aplicaciones de alto rendimiento.
  • Fácil de aprender.
  • Económico.
  • Comunidad y documentacion
  • Mejora la productividad.

Ventajas

  • Elementos gráficos que son capaces de respetar la estructura de diseño de los sistemas operativos.
  • Su acceso a funciones nativas puede realizarse a través de la reutilización del código existente en lenguajes como Objective-C, Swift y Java.
  • Ofrece widgets de interfaz de usuario desarrollados por el sistema de Google que cuentan con una alta calidad de codificación.
  • Es rápido y constante renderizado.
  • Beneficia a la productividad durante el tiempo en el que se desarrolla la aplicación.

Desventajas

  • Las aplicaciones son pesadas.
  • Las aplicaciones no son compatibles con aplicaciones web
  • Es nuevo,por lo que tnedrás que escribir muchas cosas desde el inicio.
  • Se tiene que aprender a usar Dart.

XAMARIN

Es un marco de desarrollo de aplicaciones multiplataforma alternativo para el desarrollo de aplicaciones Android e iOS y Windows con NET. Ademas permite transferir rápidamente las secuencias de comandos a otros sistemas, como Windows y macOS. La capa intermedia de Xamarin controla la conexión entre el código central del sistema y el código estándar.Tambien puede desarrollar entornos con API, backends, componentes y mucho más.

Caracteristicas

  • Útil gestor de emuladores de Google.
  • Códigos compartibles.
  • Aplicaciones de alto rendimiento.
  • Gestor para el kit de desarrollo de software (SDK) de Android.
  • Infraestructura backend versátil.

Ventajas

  • Rendimiento cercano al nativo.
  • Soporte completo de hardware
  • Tecnología de código abierto con un fuerte soporte corporativo.
  • Mantenimiento simplificado.
  • Ecositema de desarrollo completo.

Desventajas

  • El gran tamaño de la aplicación.
  • Acceso limitado a bibliotecas de código abierto.
  • Puede resultar caro.
  • Soporte ligeramente retrasado para las últimas actualizaciones de la plataforma.

REACT NATIVE

Es un marco de desarrollo de aplicaciones multiplataforma accesible que se ha convertido en la opción preferida de los programadores. Tambien facilita el desarrollo de aplicaciones móviles para Android e iOS. El principal atractivo de React Native es su mayor rapidez de desarrollo e implementación.

Caracteristicas

  • Mejor rendimiento de la aplicación.
  • Minimiza el tiempo de desarrollo.
  • Framework para Android e iOS.
  • Biblioteca fácil de usar.
  • Interfaz de usuario sencilla.

Ventajas

  • Desarrollo más rápido y barato de las Apps.
  • Código abierto.
  • Usa el lenguaje de JavaScript.
  • Multiplataforma sin problemas.
  • Arquitectura modular

Desventajas

  • Problemas de compatibilidad.
  • Actualización automática poco fiable.
  • Necesita centrarse más en el rendimiento.
  • Problemas inesperados con la adaptación a la nueva versión.

IONIC

Es un framework de aplicaciones móviles que utiliza HTML5 para desarrollar aplicaciones móviles o hibridas, el mismo que utiliza un marco front-end de AngularJS es gratis y de código abierto.

Caracteristicas

  • Ofrece un diseño limpio, sencillo y funcional.
  • Optimización del rendimiento.
  • Desarrollo multiplataforma.
  • Plazos de desarrollo de proyectos más rápidos.
  • Uniformidad de la interfaz de usuario.

Ventajas

  • Es multiplataforma.
  • Los componentes UI.
  • Usa el lenguaje de JavaScript.
  • Es libre y gratuito: .
  • Usa HTML5 y Angular JS con TypeScript.

Desventajas

  • Problemas de seguridad.
  • Ausencia de recarga en caliente.
  • Tamaño de la aplicación.
  • Sistema dependiente de plugins.
